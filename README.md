# Sales Insight Automator

AI-powered system that analyzes sales data and generates business summaries.

## Features

- Upload CSV or XLSX sales data
- AI-generated insights
- Email delivery of report
- Swagger API documentation
- Docker containerization
- CI/CD with GitHub Actions

## Tech Stack

- React
- Node.js
- Express
- Gemini API
- Nodemailer
- Docker

## Run Locally

1. Install dependencies
2. Configure `.env`
3. Run:

docker compose up --build

## API Documentation

http://localhost:5000/docs

## Workflow

Upload file → AI analysis → Email report
